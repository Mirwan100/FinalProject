        This application was made by Mirwan as part of the final project at Purwadhika Coding School. The purpose of making this application is to predict the price of a used car based on the features of the band, model, year, milage, transmission, location, and body type of the car to be sold. This application uses the Random Forest Regressor algorithm which is trained using a dataset of 80000 car sales history, the data is collected by Remko Weingarten and uploaded on atapdata.ai on 20-07-2020 for predictive analysis purposes.
